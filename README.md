# 🚀 Crypto AI Agent Dashboard

A Python-based **AI-powered cryptocurrency monitoring system** that fetches live market prices, generates AI insights, and visualizes Bitcoin trends through an interactive dashboard.

This project demonstrates **AI agents, API data pipelines, and real-time crypto monitoring using Python**.

---

# 📊 Project Overview

This system performs multiple automated steps:

1️⃣ Fetch live cryptocurrency prices from an API  
2️⃣ Store data locally in JSON format  
3️⃣ Generate AI-based summaries of market conditions  
4️⃣ Use agent logic to analyze price trends  
5️⃣ Display results in an interactive dashboard

---

# 🧠 AI Agent Workflow

The project simulates an **AI agent pipeline**:

```
Fetch Data → AI Analysis → Market Decision → Dashboard Visualization
```

# 🚀 Installation

Clone the repository

```
git clone https://github.com/YOUR_USERNAME/Agentic_AI_Learning.git
```

Navigate to the project folder

```
cd Agentic_AI_Learning
```

Install required packages

```
pip install requests streamlit
```

---

# ▶️ Running the Project

Step 1 — Fetch crypto data

```
python day1_crypto_fetch.py
```

Step 2 — Generate AI summary

```
python day2_llm_summary.py
```

Step 3 — Run the dashboard

```
streamlit run day5_crypto_dashboard.py
```
